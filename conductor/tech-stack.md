# Tech Stack

## Primary Language
- **C# (.NET 10):** Modern C# with top-tier performance and cross-platform support for building high-performance CLI games.

## Dated Notes
- **2026-04-03:** Upgraded the planned runtime from .NET 8 to .NET 10 to align with
  the installed SDK and avoid unnecessary version friction during implementation.

## Core Libraries and Frameworks
- **Spectre.Console:** A .NET library that simplifies creating beautiful, interactive, and cross-platform console applications with support for rich visuals like tables, canvases, and progress bars.
- **System.Text.Json:** The standard .NET library for high-performance JSON serialization and deserialization.

## Testing and Quality Assurance
- **xUnit:** The industry-standard testing framework for .NET applications.
- **FluentAssertions:** To provide a more expressive, human-readable syntax for test assertions.
- **Moq:** A powerful mocking library for unit testing.

## Data Persistence
- **Local JSON Storage:** A simple, human-readable JSON file for storing high scores and persistent settings.
