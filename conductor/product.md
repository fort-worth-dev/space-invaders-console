# Initial Concept

A console application of the classic arcade game Space Invaders

# Product Guide

## Project Goals
- Build a classic Space Invaders arcade game that runs entirely in the console.
- Provide a polished, stylized ASCII art experience with retro gameplay.
- Ensure high performance and smooth movement within terminal constraints.

## Core Features
- **Core Gameplay:** Responsive player movement (left/right) and firing mechanics.
- **Scoring System:** Track points for destroyed enemies with a local high-score table.
- **Wave Progression:** Increasing game speed and varied enemy patterns as the player advances through waves.
- **Stylized ASCII Art:** Detailed visuals using extended ASCII or Unicode characters for enemies, player ships, and projectiles to create a rich retro aesthetic.
- **Audio Feedback:** PC Speaker-style beeps for key events like shooting, explosions, and game-state changes.
- **High Score Persistence:** Save and load local high scores from a file.

## User Persona
- Retro gaming enthusiasts and developers looking for a classic arcade experience directly in their terminal.
